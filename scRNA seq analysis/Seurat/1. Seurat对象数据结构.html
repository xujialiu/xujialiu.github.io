Seurat
...

assay: 存储原始数据和缩放数据

  • RNA: RNA数据
    - count: 存储原始数据的稀疏矩阵
    - Dimnames: 存储barcode和基因id
    - data: 存储NormalizeData函数的LogNormalize标准化后的数据的稀疏矩阵
    - scale.data: 存储ScaleData函数对高突变基因缩放后得到的缩放数据的二维矩阵
    meta.data: 存储细胞级别的meta数据, 这些数据可以为idents函数使用, 作为绘图和计算时, 细胞的分类依据
    reduction: 存储降维数据, 包括PCA, tSNE, UMAP
  • PCA: 存储PCA后的数据, 以及PCA相关计算的数据
    - jackstraw: 存储JackStraw函数以及ScoreJackStraw函数(@overall.p.values)得到的数据
    active.ident: 每个单细胞的当前活跃标识符
    graphs: 储存基于图的聚类方法的计算结果
    Project.name: 存储数据集的名字
    version: 存储Seurat对象的版本信息
    commands: 存储对Seurat对象执行过的, 能改变Seurat对象的操作

Seurat对象是S4对象

SlotFunction
assaysA list of assays within this object
meta.dataCell-level meta data
active.assayName of active, or default, assay

reductionsA list of reductions objects
project.nameUser-defined project name (optional)
toolsEmpty list. Tool developers can store any internal data from their methods here

Assay对象级别的操作
...

assay类储存单细胞数据
对于经典单细胞实验, Seurat对象有一个assay('RNA'). assay对象储存原始数据和多种变换后的数据, 包括raw counts (@counts slot), normalized data (@data slot), and scaled data for dimensional reduction (@scale.data slot).
对于更复杂的实验, 一个对象可以包含多个assays. 每个assay又包含自己的原始数据和变换后的数据

SlotFunction
countsStores unnormalized data such as raw counts or TPMs
dataNormalized data matrix
scale.dataScaled data matrix
keyA character string to facilitate looking up features from a specific Assay
var.featuresA vector of features identified as variable
meta.featuresFeature-level meta data
